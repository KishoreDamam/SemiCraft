# Edge Detector

A single-bit any (both)-edge detector: it emits a one-clock-wide pulse on pulse one cycle after a qualifying transition on d.

_Generated by SemiCraft 0.1.0 — config hash `83dc7562b8bf`._

## Ports

### Clocking

Clock and reset for the sampling and (optional) output registers.

| Port | Direction | Description |
| --- | --- | --- |
| `clk` | input | Clock; d is sampled on the rising edge. |
| `rst_n` | input | Sync reset, active-low reset input. |

### Data

Input signal and the detected-edge pulse output (registered).

| Port | Direction | Description |
| --- | --- | --- |
| `d` | input | single-bit input signal whose edges are detected. |
| `pulse` | output | One-cycle any (both)-edge pulse, registered (asserted one cycle after the edge). |

## Configuration

- Detect: any (both) edge
- Width: 1 bit(s)
- Output: registered (one extra cycle of latency)
- Reset: sync, active-low
- Output language: SystemVerilog

## Assumptions

- A single free-running clock drives the detector; d is sampled synchronously to it.
- d is already synchronous to clk (no metastability handling here); use a CDC synchronizer upstream if d crosses clock domains.

## Limitations

- Detects transitions of d as sampled at the clock edge: a pulse on d narrower than one clock period, or between sampling edges, can be missed entirely.
- Single clock domain only; this module performs no clock-domain crossing (CDC) synchronization on d.
